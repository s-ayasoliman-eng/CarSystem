# Car Management System 

A C++ console-based car management system built using Object-Oriented Programming (OOP) concepts and file handling.  
The project simulates a simple car dealership platform where users can manage cars, customers, reviews, sales, and search operations through an interactive console interface.

---

##  Features

###  User Management
- Register and manage users
- Store user data using text files
- Handle customer information

###  Car Management
- Add new cars
- Display available cars
- Manage car details and features
- Save and retrieve car data from files

###  Search System
- Search for cars using different criteria
- Organized search functionality using dedicated classes

###  Reviews & Ratings
- Add and display customer reviews
- Manage review records using file handling

### Sales Management
- Store sales information
- Track car purchases and transactions

### User Interface
- Console-based interactive UI
- Menu-driven navigation system

## Project Structure

| File | Description |
|------|-------------|
| `main.cpp` | Main entry point of the application |
| `ApplicationManager.*` | Controls application flow |
| `car.*` | Car class and operations |
| `Customer.*` | Customer management |
| `Review.*` | Review system |
| `ReviewManager.*` | Handles reviews |
| `sales.*` | Sales management |
| `SearchCar.*` | Search functionality |
| `FileManager.*` | File handling operations |
| `UI.*` | Console user interface |
| `User.*` | User system |
| `cars.txt` | Stores car data |
| `reviews.txt` | Stores reviews |
| `sales.txt` | Stores sales records |
| `users.txt` | Stores user data |
